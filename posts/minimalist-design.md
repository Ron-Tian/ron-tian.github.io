# 极简主义设计：少即是多的力量

> "Perfection is achieved, not when there is nothing more to add, but when there is nothing left to take away."
> — Antoine de Saint-Exupéry

## 什么是极简主义？

极简主义不是简单地减少元素，而是一种**精准表达**的设计哲学。每一个保留的元素都有其存在的理由，每一个被移除的元素都是对用户的尊重。

## 核心原则

### 1. 留白即设计

留白不是"空"，而是给内容呼吸的空间。它引导视线、建立层次、传递品质感。

### 2. 聚焦核心功能

```
用户来这里的目的是什么？
→ 只保留实现这个目的所需的最少元素
```

### 3. 色彩克制

- 使用 1-2 个主色
- 大量中性色（黑、白、灰）
- 色彩用于引导注意力，而非装饰

### 4. 字体层次

通过字号、字重、行高建立清晰的信息层次：

- **大标题**：吸引注意
- **正文**：舒适阅读
- *辅助文字*：补充信息

## 在数字产品中的应用

### Apple

苹果是极简设计的典范。从硬件到软件，每一处都体现着"少即是多"的理念。

### Notion

Notion 的界面几乎没有多余的装饰，让用户专注于内容本身。

### Linear

Linear 用克制的色彩和精致的动效，打造了高效而优雅的体验。

## 极简的误区

- ❌ 极简 = 空白页面
- ❌ 极简 = 黑白灰
- ❌ 极简 = 缺少功能
- ✅ 极简 = 每个元素都有目的

## 写在最后

极简主义设计是一种**减法思维**。它要求设计师不断问自己："这个元素真的需要吗？"

当你不再添加任何东西，而是开始思考可以去掉什么时，你就理解了极简。

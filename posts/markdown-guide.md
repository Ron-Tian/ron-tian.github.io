# Markdown 完全指南

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

## 基本语法

### 标题

使用 `#` 号表示标题，数量代表层级：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

- **粗体**：使用两个星号 `**文字**`
- *斜体*：使用单个星号 `*文字*`
- ~~删除线~~：使用两个波浪号 `~~文字~~`

### 列表

**无序列表：**

- 第一项
- 第二项
  - 嵌套项
  - 嵌套项

**有序列表：**

1. 第一步
2. 第二步
3. 第三步

## 进阶语法

### 代码

行内代码：使用反引号包裹，如 `console.log()`

代码块：使用三个反引号包裹

```python
def hello(name):
    print(f"Hello, {name}!")

hello("World")
```

### 引用

> 这是一段引用文字。
>
> 可以包含多行。

### 表格

| 特性 | Markdown | Rich Text |
|------|----------|-----------|
| 可读性 | 高 | 低 |
| 可移植 | 是 | 否 |
| 版本控制 | 友好 | 不友好 |

### 链接与图片

链接：`[文字](URL)`

图片：`![描述](图片URL)`

## 实用技巧

1. **保持简洁** — Markdown 的哲学是简洁
2. **善用引用** — 突出重要内容
3. **合理分段** — 提升可读性
4. **统一风格** — 保持文档一致性

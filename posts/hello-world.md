# 你好，世界

每一段旅程都始于第一步。欢迎来到我的个人博客。

## 为什么写博客？

写作是最好的思考方式。当把模糊的想法变成文字时，你会发现思维中那些不清晰的角落。博客对我来说，是一个**整理思想、记录成长**的地方。

> "写作不是把想好的东西写下来，而是通过写来想清楚。"

## 这个博客会有什么？

作为一个综合个人博客，这里会记录：

- **技术笔记** — 编程学习、工具使用、项目经验
- **生活随笔** — 旅行见闻、日常感悟、读书心得
- **设计思考** — 产品设计、用户体验、视觉美学
- **好物分享** — 发现的好工具、好资源、好内容

## 技术实现

这个博客是一个纯静态网站，使用以下技术：

- 原生 HTML / CSS / JavaScript
- Markdown 渲染（marked.js）
- 响应式设计
